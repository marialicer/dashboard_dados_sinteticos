# Dashboard Radar 360 com dados sintéticos



###### Objetivo:



Simular um cenário realista de audiência de portal de notícias, distribuindo os dados por:



* Principais KPI's (Total de pageviews, média de pageviews e total de usuários)
* Principais editorias do site/pageviews
* Série temporal de pageviews
* Top matérias com mais pageviews
* Pageviews por origem de tráfego (search, news, discover, internal, outros)



###### Tecnologias utilizadas:



* Python (Pandas)
* Geração de dados sintéticos
* Excel
* Looker Studio

###### 

###### Lógica de modelagem



* Geração automática de 5.000 títulos únicos
* Construção de URLs com slug
* Distribuição realista de visualizações por origem
* Simulação de comportamento de tráfego digital

###### Insights gerados



No mês de dezembro/25 esses foram os principais destaques:



* Média de pageviews cresceu em 4,2% em relação ao mês anterior
* Total de usuários no mês cresceu 82,3%
* Política foi a principal editoria do site em dezembro/25
* Matérias sobre reforma tributária, políticas públicas e mercado financeiro são as que mais geram tráfego para o portal
* Google Search se consolida como a principal fonte de tráfego, com 45% dos pageviews totais
